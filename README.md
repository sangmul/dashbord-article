# 📊 Dashboard Article

Project ini adalah dashboard sederhana untuk mengelola artikel (CRUD) menggunakan:
- Backend: Golang

---

## 🚀 Features

- ✅ Create Post (Publish / Draft)
- ✅ Edit Post
- ✅ Trash Post
- ✅ List All Posts
- ✅ Preview (hanya publish)
- ✅ Pagination (limit & offset)

---

## 📦 Installation

setup Backend baseURL di .env  

Clone repository:

```bash
git clone <your-repo-url>
cd dashboard-article
npm install
npm start

http://localhost:3000
